/Köi/ A friendly companion for your
shell scripting journeys.

Köi is a minimalistic language, mainly aimed at shell scripting.

It tries to provide a familiar and ergonomic syntax for non-shell code, without compromising the ease of launching and composing programs that traditional shell languages offer.

Köi is inspired by Rust, Python, JavaScript, Lua and Bash.

Types and literals

Koi is a dinamically typed language. It has: nil, numbers, strings, booleans, vectors, dictionaries, functions and ranges. If you're familiar with Python or JavaScript you'll feel right at home.

represents the absence of a value.

There is no distinction between integers and floating points. They're the same type: number. There are places where integers are required (such as ranges) so in that case, attempting to use a number with non-zero decimal part will throw an error.

String literals can be delimited with either ' or ".

In vectors and dictionaries literals, commas are optional. They're only required in special cases where the parser is too greedy and detects a whole expression that you intended to be two separate expressions instead.

Ranges represent a range between two numbers and are used in for loops and to slice vectors. An equal sign can be used to indicate that the range is right-inclusive.

Functions literals (also called lambdas or anonymous functions) look just like regular functions but don't have a name.
